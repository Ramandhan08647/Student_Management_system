# User Manual

## Login
- Enter your username and password.
- Use the Show Password button if needed.
- Click Register to create a new user account.

## Registration
- Complete all required fields.
- Use a strong password with uppercase, lowercase, digits, and symbols.
- After registration, verify with the OTP sent to your email.

## Dashboard
- After login, users will see a dashboard with summary cards.
- Navigation will eventually include student, teacher, attendance, and report modules.

## Students
- Search by name, email, or student ID.
- Results display key student details.
