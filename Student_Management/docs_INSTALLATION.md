# Installation Guide

## Requirements
- Python 3.10 or later
- Windows, macOS, or Linux

## Install Dependencies
1. Open a terminal in the project root.
2. Run:
   ```bash
   pip install -r requirements.txt
   ```

## Run the Application
```bash
python main.py
```

## Initial Setup
- On first run, the application creates `database/asmrs.db` and `database/config.json`.
- Configure email settings in `config.py` for OTP verification.
