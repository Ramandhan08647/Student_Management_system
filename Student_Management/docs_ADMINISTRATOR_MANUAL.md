# Administrator Manual

## System Access
- Administrators have access to all modules and settings.
- Admins can manage users, students, teachers, and system configuration.

## Configuration
- Update school information and email settings in `config.py`.
- Configure SMTP settings to enable OTP email verification.

## Backups
- Backups can be stored in the `backups/` folder.
- Use the database file `database/asmrs.db` for manual restore or migration.
